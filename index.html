Qui sommes-nous ?

Nous sommes deux étudiants en développement web qui avons comme exercice de se présenter sur ce site; chacun présentant l'autre.
L'idée est d'apprendre à utiliser git et github ainsi que de collaborer sur des projets communs. Nous avons donc chacun réalisé une page représentant l'autre, pendant que l'autre réalisait une page sur nous.
